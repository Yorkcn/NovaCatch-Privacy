NovaCatch Browser Extension Privacy Policy

NovaCatch is a local browser resource detection extension. It helps users identify downloadable resources on pages they visit and send selected resources to the NovaCatch desktop application running on the same computer.

Effective date July 15, 2026
Applies to NovaCatch Browser Extension
Data destination Local browser storage and local desktop app

English

Single purpose

NovaCatch detects downloadable resources on user-visited pages and lets the user send selected resources or user-clicked browser download links to the local NovaCatch desktop application.

Data processed by the extension

The extension may process the following data to provide its single purpose:

  • Page URLs and downloadable resource URLs.
  • Resource type, file extension, MIME type, file size, filename candidate, source page, and tab metadata.
  • Request metadata such as Content-Type, Content-Length, and Content-Disposition.
  • Limited request headers needed for a user-authorized local download, such as Referer, Origin, and Cookie when required by the source site.
  • Extension settings, blocked-site rules, whitelist rules, Feishu/Lark sniffing settings, and short-lived per-tab resource lists.

How data is stored and transmitted

NovaCatch stores extension settings and short-lived resource lists in Chrome extension storage on the user's device. When the user sends a resource to the desktop app, or when browser-download capture is enabled, the extension transmits the selected download URL and required metadata to the local NovaCatch desktop endpoint at localhost:15151.

No sale, advertising, or cloud tracking

  • NovaCatch does not sell user data.
  • NovaCatch does not use user data for advertising.
  • NovaCatch does not send browsing history or resource URLs to a NovaCatch cloud service.
  • NovaCatch does not transfer user data for purposes unrelated to the extension's single purpose.

User controls

Users can control the extension's behavior by using the extension settings:

  • Disable browser-download capture.
  • Configure blocked sites or whitelist mode.
  • Disable Feishu/Lark document sniffing.
  • Clear captured resource data from the extension UI.

Legal and site-permission boundary

NovaCatch is intended only for downloading resources that the user owns, controls, or is authorized to download. It must not be used to bypass DRM, paywalls, account permissions, site download/export permissions, or copyright restrictions.

Contact

For privacy questions, use the support contact or support page listed in the NovaCatch Chrome Web Store listing.

中文

单一用途

NovaCatch 的单一用途是识别用户当前访问网页中的可下载资源，并将用户选择的资源或用户点击的下载链接发送到本机运行的 NovaCatch 桌面客户端。

扩展会处理的数据

为实现上述单一用途，扩展可能会处理以下数据：

  • 页面 URL 和可下载资源 URL。
  • 资源类型、文件扩展名、MIME 类型、文件大小、候选文件名、来源页面和标签页元数据。
  • Content-Type、Content-Length、Content-Disposition 等响应元数据。
  • 用户授权下载所需的有限请求头，例如来源网站要求的 Referer、Origin 和 Cookie。
  • 扩展设置、站点屏蔽规则、白名单规则、飞书/Lark 嗅探设置和短期的按标签页资源列表。

数据如何存储和传输

NovaCatch 会把扩展设置和短期资源列表保存在用户设备上的 Chrome 扩展本地存储中。当用户主动发送资源，或启用浏览器下载接管功能时，扩展会把选中的下载 URL 和必要元数据发送到本机 NovaCatch 桌面客户端的本地端点 localhost:15151。

不出售、不广告、不云端跟踪

  • NovaCatch 不出售用户数据。
  • NovaCatch 不将用户数据用于广告。
  • NovaCatch 不会把浏览历史或资源 URL 发送到 NovaCatch 云服务器。
  • NovaCatch 不会为了与扩展单一用途无关的目的转移用户数据。

用户控制

用户可以在扩展设置中控制以下行为：

  • 关闭浏览器下载接管。
  • 配置站点屏蔽规则或白名单模式。
  • 关闭飞书/Lark 文档嗅探。
  • 在扩展界面中清理已捕获的资源数据。

法律与网站权限边界

NovaCatch 仅用于下载用户拥有、控制或被授权下载的资源。NovaCatch 不应用于绕过 DRM、付费墙、账号权限、网站下载/导出权限或版权限制。

联系

如有隐私相关问题，请使用 NovaCatch Chrome Web Store 商品详情中提供的支持联系方式或支持页面。
